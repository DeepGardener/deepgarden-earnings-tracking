# 示例说明：联影医疗（688271.SH）定期报告阅读整理

本文件夹是 **DeepGarden 定期财报跟踪** Skill 的**真实端到端产出示例**，由该 Skill 依据一套
「经营 / 资产质量 / 盈利质量 / 值得重视的信息」四维度 22 项关注点的阅读框架自动生成。

目的：让使用者（安装者）直观看到该 Skill 最终产出**长什么样**——文件结构、笔记深度、页码引用方式。

## 文件结构（与 Skill 实际产出一致）

```
example-uimages/
├─ 00-联影医疗索引.md              # 公司索引：跨年财务快照表 + 主线叙述
└─ 阅读整理/
   ├─ 2023年报-阅读整理.md
   ├─ 2024年报-阅读整理.md
   ├─ 2025年报-阅读整理.md
   └─ 2026一季报-阅读整理.md
```

## 阅读要点

- 每份阅读 MD 都**严格覆盖**提示词框架的四维度（经营 / 资产质量 / 盈利质量 / 值得重视的信息），
  共计 22 个关注点逐条展开，并标注来源页码 `[Pxx]`。
- 笔记中的 `[Pxx]` 页码引用**指向原始 PDF 报告对应页**；本示例**不含原始 PDF**
  （上市公司定期报告受版权与体积限制，未随 Skill 分发），不影响对产出效果的理解。
- 联影医疗主线速览（来自索引）：
  - 2023：营收 114 亿(+23.5%)、净利 19.7 亿(+19.2%)，但经营现金流仅 1.33 亿、应收激增，盈利含金量偏低。
  - 2024：业绩低谷——营收 −9.7%、净利 −36.1%、经营现金流 −6.19 亿（政策节奏错位致需求积压）。
  - 2025：强势反转——营收 138 亿(+34%)、净利 18.7 亿(+48%)、现金流由负转正至 26.8 亿。
  - 2026Q1：收入 +17.3% 延续修复，但扣非 −1.75%、盈利质量边际走弱，需持续跟踪应收与现金流。
